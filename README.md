# ApplePieRecipe
This project is a simple Apple Pie Recipe webpage built using HTML5.  The purpose of the project was to practice structuring webpages using semantic HTML elements while organizing content clearly and effectively.
# Apple Pie Recipe – HTML Project
## Skills Used

* HTML5
* Semantic HTML Structure
* Lists (Ordered & Unordered)
* Images & Alt Text
* Hyperlinks
* Footer Elements
* Content Organization

## Features

* Apple pie recipe webpage
* Ingredient list
* Step-by-step instructions
* Embedded image
* External LinkedIn profile link
* Clean HTML page structure
